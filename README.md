# Delay Plugin
**133.385 Music Software Development 2 - Second Assessment**

*Designed & written by Edward Lilley & Jordan La'ulu using JUCE v5.4.3*


## Features
* Graphical User Interface
* Wet(%), Delay Time(msecs) and Feedback(%) sliders
* Editable text boxes
* Smoothed Values
* Automation
* Save & Recall State
